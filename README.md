# README - Backend ONG Lar Miau 🐾
Este projeto faz parte do trabalho para a disciplina de Programação WEB e foi desenvolvido para a ONG Lar Miau, que se dedica ao resgate, cuidado e apadrinhamento de gatos. O backend é responsável por gerenciar duas tabelas principais: gatos e padrinhos, e fornece funcionalidades essenciais para as operações da ONG, como adição, exclusão, atualização e obtenção de dados.

## Funcionalidades

### CRUD Completo:

GET: Obtenção de dados sobre os gatos e padrinhos.

POST: Adição de novos gatos e padrinhos.

PUT: Atualização das informações de um gato ou padrinho.

DELETE: Remoção de gatos ou padrinhos do sistema.

Rotas organizadas por recursos:

/gato: Operações relacionadas aos gatos.

/padrinho: Operações relacionadas aos padrinhos.

Integração com o frontend: O backend fornece dados e serviços para a aplicação React responsável pela interface de apadrinhamento e gerenciamento de gatos.

## Tecnologias Utilizadas
Node.js: Ambiente de execução JavaScript no servidor.

Express: Framework para construção de APIs rápidas e escaláveis.

MySQL/PostgreSQL: Banco de dados relacional usado para armazenar informações sobre gatos e padrinhos.

JavaScript: Linguagem de programação utilizada para toda a lógica backend.

## Instalação e Execução

### Pré-requisitos
Node.js

NPM

Banco de dados MySQL/PostgreSQL configurado
